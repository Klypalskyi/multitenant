# `@multitenant/react`

**React context** and hooks for **tenant**, **market**, and **feature flags**, fed from server-provided props (or your own **`ResolvedTenant`**). Pairs with **`@multitenant/next-app`** or any app that can serialize resolution into the tree.

## Install

```bash
npm install @multitenant/react react
```

Peer: **`react`** ≥ 18.

## Main APIs

- **`TenantProvider`**
- **`useTenant`**, **`useMarket`**, **`useTenantConfig`**, **`useTenantFlag`**, and related selectors
