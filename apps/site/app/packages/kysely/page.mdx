# `@multitenant/kysely`

Thin **Kysely** + **`pg`** helpers: **`getOrCreateTenantNodePgPool`**, **`getTenantNodePgKysely`**, **`createNodePgKysely`** — per-tenant **Kysely** from **`ResolvedTenant`**. Postgres-oriented; resolution stays in **`@multitenant/core`**.

## Install

```bash
npm install @multitenant/kysely kysely pg
```

## Main APIs

- **`getOrCreateTenantNodePgPool`**, **`getTenantNodePgKysely`**, **`createNodePgKysely`**
