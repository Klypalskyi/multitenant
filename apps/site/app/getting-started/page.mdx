# Getting started

1. **Scaffold** ‚Äî `npx multitenant init` writes `tenants.config.json` and optional framework stubs (`--framework next-app | next-pages | express`).

2. **Validate** ‚Äî `npx multitenant check` runs the same Zod validation as `loadTenantsConfig`.

3. **Wire your framework** ‚Äî install `@multitenant/next`, `@multitenant/express`, etc., and follow the checklist for your stack.

4. **Local multi-host** ‚Äî run your app on one port (e.g. 3000) and **`multitenant dev --target http://localhost:3000 --port 3100`** so tenant hosts hit the proxy.

For the full guide (async config, Edge vs Node, identity), see the repo:
