**Node-only** **AsyncLocalStorage** scope for the current tenant (and related context): **`runWithTenantScope`** so downstream DB layers see **`tenantKey`** / **`environment`** without threading arguments everywhere. Includes Postgres-oriented pool helpers and a **bounded pool cache**; **not** for Edge runtimes.

## Install

```bash
npm install @multitenant/database
```

## Main APIs

- **`runWithTenantScope(store, fn)`**
- Postgres pool helpers aligned with **`ResolvedTenant`** (see package **`README`** / exports)
