The **`multitenant`** binary: scaffold **`tenants.config.json`**, run the **local dev proxy**, validate config, and print resolution/debug output.

## Install

```bash
npm install -D @multitenant/cli
# or run without installing: npx @multitenant/cli ‚Ä¶
```

## Commands

- **`multitenant init`** ‚Äî generate starter config and docs pointers
- **`multitenant dev`** ‚Äî proxy mode with tenant-by-host resolution and config hot reload (uses **`@multitenant/dev-proxy`**)
- **`multitenant check`** ‚Äî validate **`tenants.config.json`** (via **`@multitenant/config`**)
- **`multitenant print`** ‚Äî dump effective config / resolution hints for debugging
