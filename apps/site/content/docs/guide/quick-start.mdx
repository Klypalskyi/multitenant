## Commands

```bash
npx @multitenant/cli init --force
npx @multitenant/cli check
```

Then wire **`@multitenant/next-app`**, **`@multitenant/express`**, or another adapter — see [Getting started](/docs/guide/getting-started) and [Next.js](/docs/packages/next).

For the mental model and pitfalls first, read [Why Multitenant?](/docs/why-multitenant).
